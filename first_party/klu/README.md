# Kotlin Lightweight Upgrade (KLU)

General helpers and utilities for Kotlin.

## Release

Released to [Maven Central](https://search.maven.org/artifact/com.jackbradshaw/klu).

## Purpose

This package provides a set of general-purpose helpers and utilities for Kotlin development.

## Contents

KLU contains various packages:

- [Concurrency](concurrency)
- [Flows](flow)

## Usage

The packages may be used as library dependencies.
