# Model

Universal data models that apply across all domains.

## Release

Not released to third party package managers.

## Contents

This package provides the following data models:

- [count](/first_party/model/count): A model for representing a discrete quantity.
- [frequency](/first_party/model/frequency): A model for representing a rate of occurrence.
