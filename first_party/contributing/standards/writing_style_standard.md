# Writing Style Standard

The standard for writing style in this repository.

## Scope

All documentation and comments in this repository must conform to this standard; however, the contents of
[third_party](/third_party) are exempt as they originate from other repositories.

## Grammar

Grammar must adhere to the following requirements:

- American English must be used, except where domain conventions dictate otherwise (e.g. referencing
 a `Colours` object form a third party package).
- All spelling must be correct.
- Commas must be omitted after abbreviations (e.g. "etc" instead of "etc.").

These requirments ensure consistency and legibility.
