# Standards

The standards for contribution to this repository.

## Purpose

These standards ensure the contents of the repository are universally accessible and useful. They
must be satisfied prior to submission to the main branch.

## Contents

This package contains the following standards:

- [Standard for Standard Documents](/first_party/standards/standard_for_standards.md)
- [General Documentation Standard](/first_party/standards/general_documentation_standard.md)
- [README Standard](/first_party/standards/readme_standard.md)
- [Markdown Standard](/first_party/standards/markdown_standard.md)
- [General Code Block Standard](/first_party/standards/general_code_block_standard.md)
- [Shell Code Block Standard](/first_party/standards/shell_code_block_standard.md)
- [Starlark Code Block Standard](/first_party/standards/starlark_code_block_standard.md)
- [TODO Standard](/first_party/standards/todo_standard.md)

## Usage

The standards may be passed to artificial intelligence for automated content generation and linting.

## Issues

Issues relating to this package and its subpackages are tagged with `standards`.

Open to contributions from third parties.
